# 🩺 Medical Data Analysis – Diabetes Dataset

## 📌 Project Overview
This project focuses on analyzing **medical patient data** to discover insights about health risk factors.  
The dataset used is the **PIMA Indian Diabetes dataset**, which contains diagnostic measurements to predict the likelihood of diabetes in patients.

The aim of this project is to:
- Perform **data cleaning** and **preprocessing**
- Conduct **exploratory data analysis (EDA)**
- Generate **visualizations and insights**
- (Optional) Build a simple **predictive model**

## 📊 Dataset Information
- **Source:** [PIMA Indian Diabetes Dataset (Kaggle/UCI)](https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database)  
- **Rows:** 768  
- **Columns:** 9  

**Columns include:**
- Pregnancies
- Glucose
- BloodPressure
- SkinThickness
- Insulin
- BMI
- DiabetesPedigreeFunction
- Age
- Outcome (0 = Non-diabetic, 1 = Diabetic)
